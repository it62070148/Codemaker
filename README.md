# Introdution
### สวัสดีครับวันนี้พวกผมกลุ่มของผมได้ทำ โปรแกรม ขึ้นมาซึ้งเป็นโปรแกรมเข้ารหัสเข้ารหัสและถอดรหัสครับชื่อว่า Codemaker ซึ่ง โปรแกรมนี้ เหมาะกับการเข้ารหัสข้อมูลสำคัญต่างๆ เช่นพวก รหัสเข้าแอพพลิเคชั่นต่าง หรือจะต้องการเข้ารหัสข้อความและต้องการส่งให้ผู้รับโดยอยากต้องการให้ท่านกับผู้รับรู้กันอย่างเดียว                              .
